# Robi Gadget Shop

Static Astro storefront for Robi Gadget Shop.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4321/`.

## Build

```bash
npm run build
```

The production site is configured for GitHub Pages at:

```txt
https://polodev.github.io/robigadgetshop/
```
